- **RAG Chatbot** (github - https://github.com/rupamdebnath627/Chatbot)  
(Python, Langchain, Gradio)  
-- This a RAG based chatbot built in Python with the help of LangChain. The primary use of this app is that you can feed your documentation website in it. It will scrape though it. It will make embeddings of the data and store them in a vector database. Later when you send a query to the bot, it will first fetch top 5 similar embeddings and then send it to the public AI LLM tools like Gemini or Chatgpt. These LLM tools will be able to answer your queries with the help of the top 5 similar data that it fetched from the Vector DB.
